# 📊 Trader Performance vs Market Sentiment Analysis

## 🚀 Overview

This project analyzes how Bitcoin market sentiment (Fear/Greed Index) influences trader performance using historical trading data.

## 📂 Datasets Used

* Bitcoin Fear & Greed Index
* Historical Trader Data (Hyperliquid)

## 🔍 Key Insights

* Traders have **highest win rates during Extreme Greed (~55%)**
* **Largest trade sizes occur during Fear**, indicating risk-taking behavior
* Profitability shows **high volatility across all sentiments**
* Weak correlation between sentiment and profit → sentiment alone is not sufficient

## 📈 Key Findings

* Extreme Fear → Lowest win rate (~29%)
* Fear → Highest average trade size (~5700 USD)
* Greed phases → Higher participation but unstable profits

## 🧠 Conclusion

Trader psychology plays a more critical role than market sentiment alone. Combining sentiment with risk management strategies is essential for consistent profitability.
